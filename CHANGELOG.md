# Changelog

Todos los cambios notables en este proyecto serán documentados en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
y este proyecto adhiere a [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [5.0.1] - 2024-02-16

### Added
- Nuevo diseño cyberpunk con animaciones vivas
- Sistema de partículas interactivo
- Efecto Matrix en el background
- Scanlines y efectos CRT
- Terminal de comandos funcional
- Dashboard de salud digital interactivo
- Score circular animado
- Progress bars con efectos de brillo
- 6 servicios con cards animadas
- Sección de proceso paso a paso
- Carrusel de testimonios
- Formulario de contacto funcional
- Múltiples CTAs (WhatsApp, Email, Calendly)
- Navegación sticky con scroll progress
- Footer completo con links SEO
- Meta tags optimizados para SEO
- Schema.org structured data
- Loading screen animada
- Responsive design completo

### Changed
- Rediseño completo de la interfaz
- Mejoras significativas en UX/UI
- Optimización de performance

### Fixed
- Problemas de navegación móvil
- Optimización de animaciones

## [4.0.0] - 2024-01-15

### Added
- Versión inicial de RON3IA
- Dashboard básico
- Consola de comandos
